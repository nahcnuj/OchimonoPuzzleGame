# 落ち物パズルゲーム

2013 年高専祭展示作品です。

Release/puzzle.exe を実行すると遊べます。

## 製作環境
- Windows 7 Home Premium SP1 (32 bit)
- Microsoft Visual C++ 2010 Express Edition
- DirectX 9 (June 2010)
- DX ライブラリ使用

## 遊び方

### 操作方法
- キーボードのみ対応しています。ゲームパッドでは操作できません。
- ←: ブロックを左へ移動
- →: ブロックを右へ移動
- ↑: ブロックを下まで落下
- ↓: ブロックを時計回りに 90 度回転
- ESC: ゲームを終了

### 遊び方
ゲームを起動するとタイトル画面が表示されます。
この画面でスペースキーを押すとゲーム画面に切り替わり、ゲームが始まります。
ルールについては、次の「ルール」を見てください。タイトル画面にも簡単な説明が表示されます。
ゲームオーバーになるとリザルト画面に切り替わります。
この画面でスペースキーを押すとタイトル画面に戻ります。
また、キーボード左上の ESC を押すとゲームを終了することができます。

### ルール
上から落ちてくるブロックを操作して、縦か横に `1 + 3` のような式を作ると、そのブロックが消えます。
このとき、式の計算結果がスコアに加えられます。
式の判定は右方向と下方向に行われます。
ブロックの消去は式の判定が終了したあとにまとめて行われます。
`1 + 2 × 3` のような場合は、`(1 + 2) + (2 × 3) = 9` がスコアに加えられます。
引き算で負になる場合は、スコアも減少します。例：スコア 30 のときに `3 - 7` を作ると、`30 + (3 - 7) = 26` になります。
割り算で小数になる場合は、小数第 2 位が切り捨てられてスコアに加えられます。
スコアが 250 増えるごとにレベルが 1 上がり、落下スピードが少し速くなります。
一番上の段までブロックが積まれてしまうとゲームオーバーです。

### スコアランキング
ゲームオーバーになると、あなたのスコアとランキングが表示されます。
ランキングは 5 位までのスコアを保存しています。
